Web Development Topics

HTML • CSS • JavaScript

HTML Topics

1. HTML Headings

HTML headings are used to define headings on a webpage. They range from <h1> to <h6>. 

<h1>Main Heading</h1>
<h2>Sub Heading</h2>

2. HTML Paragraph

Paragraph tag <p> is used to write normal text content. 

<p>This is a paragraph</p>

3. HTML Links

Anchor tag is used to create links between pages. 

<a href="index.html">Go to Home</a>

CSS Topics

1. CSS Color

CSS color property is used to set text color. 

p {
  color: red;
}

2. CSS Background Color

Background-color sets the background of an element. 

div {
  background-color: black;
}

3. CSS Flexbox

Flexbox is used to create responsive layouts easily. 

.container {
  display: flex;
  gap: 20px;
}

JavaScript Topics

1. JavaScript Variables

Variables are used to store data values. 

let name = "Ali";
const age = 20;

2. JavaScript Function

Functions perform a specific task when called. 

function greet() {
  alert("Hello World");
}

3. JavaScript Button Click

JavaScript can respond to user actions like button clicks. 

<button onclick="sayHi()">Click</button>

<script>
function sayHi() {
  alert("Hi!");
}
</script>
