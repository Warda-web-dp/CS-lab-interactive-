# CS Lab Interactive

**CS Lab Interactive** ek educational web project hai jo students ke liye algorithms aur data structures ko interactive aur visual way me samjhata hai.  

---

## 🔹 Features

- **Visualize Algorithms** (Bubble Sort, etc.)  
- Interactive quizzes  
- Topics page for quick reference  
- Responsive design (mobile-friendly)  
- Clean and simple interface
